HTML Cheat Sheet

Quick Reference

General Table List

General Tags

Tag Name Description
<!DOCTYPE> Document Type Defines the document type
<html> HTML Defines an HTML document
<head> Head Contains metadata/information for the document
<title> Title Defines a title for the document
<body> Body Defines the document's body
<h1> Heading Defines a large heading
<p> Paragraph Defines a paragraph
<br> Break Inserts a single line break
<hr> Horizontal Rule Defines a thematic break
<!--...--> Comment Defines a comment

General Attributes

Attribute Name Description
id ID Specifies a unique id for an element
class Class Specifies a class name for an element
style Style Specifies an inline style for an element
title Title Specifies extra information about an element
lang Language Specifies the language of the element's content
dir Direction Specifies the text direction of the element's content
href Hyperlink Reference Specifies the URL of the page the link goes to
src Source Specifies the URL of the image
alt Alternative Text Specifies an alternate text for an image

Table Tags

Tag Name Description
<table> Table Defines a table
<th> Table Header Defines a header cell in a table
<tfoot> Table Footer Defines a footer in a table
<tr> Table Row Defines a row in a table
<td> Table Data Defines a cell in a table

Table Attributes

Attribute Name Description
border Border Specifies the width of the border
cellpadding Cell Padding Specifies the space between the cell wall and the cell content
cellspacing Cell Spacing Specifies the space between cells
width Width Specifies the width of a table
height Height Specifies the height of a table
align Align Specifies the alignment of a table
bgcolor Background Color Specifies the background color of a table
bordercolor Border Color Specifies the color of the border
colspan Column Span Specifies the number of columns a cell should span
rowspan Row Span Specifies the number of rows a cell should span

List Tags

Tag Name Description
<ul> Unordered List Defines an unordered list
<ol> Ordered List Defines an ordered list
<li> List Item Defines a list item
<dl> Definition List Defines a definition list
<dt> Definition Term Defines a term/name in a definition list
<dd> Definition Description Defines a description of a term/name in a definition list

List Attributes

Attribute Name Description
type Type Specifies the type of list
start Start Specifies the start value of an ordered list
value Value Specifies the value of a list item
compact Compact Specifies that the list should render compactly
